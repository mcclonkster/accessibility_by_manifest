## Why This File Is Frozen

This root plan captured an earlier useful direction, but it is now stale in ways
that can mislead future agents:

- it says normalization/IR is the biggest missing layer, but a shared normalized
  accessibility model and PDF bridge now exist
- it treats the review-decision loop as the next major step, but review
  decisions, templates, OCR recovery, table decisions, and figure decisions have
  since been implemented or partially implemented
- it does not reflect the current accessible HTML sibling output path
- it does not reflect the current run-record, operator-guide, and explainer work
- it predates the current claim-policy and AGENTS.md split

## What To Preserve From This File

The useful historical point remains:

```text
v0.1.0 is not perfect accessible PDF generation.
It is a reproducible local workflow that can extract evidence, normalize
structure, create review tasks, accept review decisions, write a limited tagged
draft, validate internally, and honestly stop unless finalization is justified.
```

Do not update this file with new execution state. Update the active plan files
listed above.
