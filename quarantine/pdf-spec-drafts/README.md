# PDF Spec Drafts

This directory contains draft and historical specification material for the
PDF accessibility workflow.

These documents remain useful for implementation history, terminology origin,
and design intent, but they are not the final authority when current files say
something different.

Use this directory for:

- draft rationale
- historical implementation intent
- terminology archaeology

Do not use this directory as:

- the final v0.1.0 source of truth
- the canonical node-name reference
- the governing workflow contract

Current authority order:

1. `AGENTS.md`
2. `project-file-structure-md/`
3. active code under `src/pdf_accessibility/`
4. active tests under `tests/pdf_accessibility/`

If a draft here conflicts with current planning files, code, or tests, prefer
the current materials and treat the draft as superseded history.
