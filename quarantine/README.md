# Quarantine

This directory holds material removed from the active guidance surface without
deleting it.

Quarantined files may contain useful history, provenance, or draft thinking, but
they are not current instructions. If a quarantined file conflicts with
`AGENTS.md`, `README.md`, `docs/design/`, `plans/`, or active code/tests, the
active sources win.

## Contents

- `docs-chats/`: historical chat transcripts and handoff notes.
- `pdf-spec-drafts/`: superseded PDF workflow draft specs.
- `project-guidance/`: older project-control, cleanup, and planning guidance.
- `reference/`: old under-labeled reference material. The runtime PDF manifest
  schema was promoted back into the package under
  `accessibility_by_manifest/manifest/schemas/`.
- `system-files/`: non-project filesystem artifacts kept out of active docs.

Internal links inside quarantined files may still point at old paths. Treat them
as historical context unless promoted deliberately.
