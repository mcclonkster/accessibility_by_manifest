# Quick checks

Check every box to pass.

- [ ] This is the exact `.docx` that will be posted or shared.
- [ ] Word Accessibility Checker was run on the final file.
- [ ] The document has a clear title and built-in headings.
- [ ] Important text is real text, not only text inside images.
- [ ] Lists, links, and tables are understandable and structured correctly.
- [ ] Meaningful images, charts, and screenshots have usable descriptions.
- [ ] The document still makes sense without the original slide layout.
- [ ] Color is not the only way meaning is conveyed.
- [ ] Script-generated warnings were reviewed and resolved or accepted.
- [ ] The file was checked again after final edits.

## Fail conditions

Fail the file if any of these are true.

- [ ] Missing or weak descriptions still block understanding.
- [ ] Important meaning depends on layout, arrows, grouping, or color only.
- [ ] The document contains image-only text that should be real text.
- [ ] Reading order is confusing.
- [ ] The file still has unresolved checker issues that matter.
- [ ] The script sidecar flagged problems that were not reviewed.

## Escalate if needed

- [ ] Needs accessibility specialist review
- [ ] Needs legal/compliance review
- [ ] Needs review of a later PDF or posted version
