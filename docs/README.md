# Documentation

This directory is for project documentation that should remain useful to
contributors.

## Active

- `design/`: current architecture and design references.

## Reference

- `reference/official/`: official standards, law, agency, and accessibility
  source captures.
- `reference/pdf/`: PDF/PDF accessibility technical reference captures.

Reference material is evidence, not execution guidance. If a reference conflicts
with `AGENTS.md`, `plans/`, or active code/tests, treat the active project
sources as governing.
